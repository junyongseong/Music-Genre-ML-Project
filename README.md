# Music Genre ML Project

MBTI 기반 음악 장르 분류 및 추천 시스템

## 프로젝트 소개

사용자의 음악 취향을 분석하여 맞춤형 음악을 추천하는 웹 애플리케이션입니다. 8가지 질문을 통해 사용자의 성향을 분석하고, MBTI 유형을 기반으로 음악 장르를 추천합니다.

## 주요 기능

- 8가지 질문을 통한 음악 취향 분석
- MBTI 유형 기반 음악 추천
- 다양한 음악 장르 지원 (록, 발라드, 트로트, 재즈, 랩, K-POP, 팝, 클래식 등)

## 기술 스택

- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask
- 향후 추가 예정: 머신러닝 모델, 음악 장르 분류 알고리즘

## 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/junyongseong/Music-Genre-ML-Project.git
cd Music-Genre-ML-Project

# 필요한 패키지 설치
pip install flask

# 애플리케이션 실행
python app.py
or
py app.py
```

실행 후 브라우저에서 `http://localhost:5000`으로 접속하세요.

## 사용 방법

1. 웹 애플리케이션에 접속
2. 8가지 질문에 답변 (환경, 선호 장르, 악기, 집중 요소, 감정, 활동, 장르 선호도, 음악 발견 방법)
3. 결과 페이지에서 MBTI 유형과 맞춤 음악 추천 확인

## 향후 계획

- 머신러닝 모델 통합
- 실제 음악 데이터베이스 연동
